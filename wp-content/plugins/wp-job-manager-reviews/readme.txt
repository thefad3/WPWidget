=== WP Job Manager - Reviews ===
Contributors: Astoundify
Requires at least: 4.1 
Tested up to: 4.2.1
Stable tag: 1.3.0
License: GNU General Public License v3.0

Adds the ability to customize the review process by adding multiple review categories and controlling the star count.

= Documentation =

Usage instructions for this plugin can be found in the Listify documentation: [http://docs.astoundify.com/category/762-wp-job-manager---reviews](http://docs.astoundify.com/category/762-wp-job-manager---reviews).

= Support Policy =

We will happily patch any confirmed bugs with this plugin, however, We will not offer support for:

1. Customisations of this plugin or any plugins it relies upon
2. CSS Styling (this is customisation work)

If you need help with customisation you will need to find and hire a developer capable of making the changes.

== Installation ==

To install this plugin, please refer to the guide here: [http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation)
